# ماشین حساب نساجی — نسخه وب

ابزار محاسباتی مبتنی بر مرورگر برای صنعت نساجی، توسعه‌یافته توسط گروه **پارسیانیک**. این پروژه به صورت یک فایل HTML خودکفا نوشته شده، بدون فریم‌ورک، بدون مرحله Build، و بدون وابستگی خارجی به جز فونت Vazirmatn.

🌐 **آدرس زنده:** [calc.parsianik.com](https://calc.parsianik.com) *(آدرس Cloudflare Pages خود را جایگزین کنید)*

---

## ویژگی‌ها

- ۱۵ فرمول در ۴ دسته‌بندی فرآیند تولید
- رابط کاربری فارسی RTL با فونت وزیرمتن
- طراحی واکنش‌گرا — منوی ثابت در دسکتاپ، منوی کشویی در موبایل
- دکمه همبرگر با انیمیشن باز/بسته در صفحات تا ۷۲۰ پیکسل
- بستن منو با کلیک روی پس‌زمینه تاریک
- محاسبه آنی با پشتیبانی از کلید Enter
- اعتبارسنجی ورودی با هایلایت خطا
- نمایش انیمیشن‌دار نتیجه
- بدون نصب — در هر مرورگر مدرنی اجرا می‌شود

## دسته‌بندی‌ها و فرمول‌ها

### ۱. ذوب ریسی (Melt Spinning)
| فرمول | ورودی‌ها |
|---|---|
| وزن تئوری تولید در دقیقه | سرعت، تعداد سر نخ، نمره به دنیر |
| وزن تئوری تولید در زمان فعالیت | وزن تئوری در دقیقه، زمان کارکرد ماشین |
| وزن تئوری محصول تولیدی | درصد مستربچ، وزن چیپس، درصد روغن |
| تعداد بوبین کامل تئوری | وزن تئوری محصول، وزن بوبین کامل |
| وزن ضایعات کل | ضایعات ثابت، وزن ضایعات خط |
| راندمان تولید | وزن محصول نهایی واقعی، وزن تئوری در زمان فعالیت |

### ۲. تکسچره (Texturing)
| فرمول | ورودی‌ها |
|---|---|
| وزن تئوری تولید (کیلوگرم) | تعداد پوزیشن، مدت زمان تولید، سرعت تولید، نمره نخ خروجی |
| نمره نخ خروجی (دنیر) | نمره نخ ورودی، نسبت کشش، درصد روغن، درصد تغذیه کش، نمره کش |
| راندمان تولید (%) | میزان نخ تولید شده واقعی، وزن تئوری تولید |
| راندمان کیفی (%) | میزان نخ تولید شده واقعی، میزان درجات کیفی تولید شده |

### ۳. تابندگی (Twisting)
| فرمول | ورودی‌ها |
|---|---|
| تعداد تاب در متر | سرعت برداشت، سرعت تویستر |
| نمره نخ خروجی | نمره نخ ورودی، تعداد تاب در متر |

### ۴. بافندگی (Weaving)
| فرمول | ورودی‌ها |
|---|---|
| وزن گرد بر (گرم/متر²) | تراکم پود، نمره تار، نمره پود، تراکم تار |
| وزن متر طول (گرم/متر) | عرض، وزن گرد بر |
| تعداد رشته نخ چله‌پیچی | ظرفیت قفسه، تعداد سرنخ کل |

## استقرار روی Cloudflare Pages

۱. به [pages.cloudflare.com](https://pages.cloudflare.com) بروید
۲. یک پروژه جدید بسازید → Direct Upload
۳. فایل `textile_calculator.html` را آپلود کنید
۴. تمام — Cloudflare یک آدرس دائمی عمومی می‌دهد

برای به‌روزرسانی، کافی است نسخه جدید فایل را آپلود کنید. نیازی به Pipeline نیست.

---

## تعبیه در Google Sites

۱. ابتدا فایل را روی Cloudflare Pages مستقر کنید
۲. در Google Sites → Insert → Embed → By URL
۳. آدرس Cloudflare Pages را وارد کنید
۴. اندازه iframe را تنظیم کنید

> توجه: Google Sites کدهای JavaScript را از HTML مستقیم حذف می‌کند. روش iframe تنها راه صحیح است.

## سفارشی‌سازی

| چه چیزی | کجا در فایل |
|---|---|
| آیکون همبرگر (موبایل) | `<button class="hamburger">` — تگ‌های `<span>` را با `<img>` جایگزین کنید |
| لوگوی برند در منوی کناری | `<div class="brand-mark">` — SVG را با `<img>` جایگزین کنید |
| نام برند | `<div class="brand-name">` |
| زیرنویس (تعداد فرمول‌ها) | `<div class="brand-sub">` |
| متن صفحه خوش‌آمدگویی | `<div class="welcome">` |
| خط اعتبار پایین صفحه | تگ `<h5>` داخل `.welcome` — با `position: fixed` به پایین-چپ پین شده |
| Favicon | `<link rel="icon" type="image/png" href="favicon.png">` را در `<head>` اضافه کنید |
| پالت رنگ | متغیرهای CSS در `:root` (ابتدای بلوک `<style>`) |
| فرمول‌ها | آرایه `const DATA = [...]` در بلوک `<script>` |

## افزودن فرمول جدید

داخل آرایه `DATA` در بلوک `<script>`، یک ورودی به دسته‌بندی مرتبط اضافه کنید:

```javascript
{ title: 'نام فرمول',
  unit: 'واحد خروجی',
  inputs: [
    { id: 'a', label: 'متغیر اول' },
    { id: 'b', label: 'متغیر دوم' },
  ],
  calc: v => v.a * v.b   // فرمول شما اینجا
},
```

منوی کناری و کارت فرمول به صورت خودکار تولید می‌شوند — نیازی به تغییر دیگری نیست.
